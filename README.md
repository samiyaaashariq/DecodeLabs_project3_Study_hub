# 📚 Study Resources Hub

A simple and modern **React-based Study Resources Manager** that helps students organize, search, and access their study materials in one place.

Built as an internship project to demonstrate frontend development skills, state management, and clean UI design.

## ✨ Features

- ➕ Add study resources (Title, Subject, Type, URL)
- 🔍 Search resources instantly
- 🎯 Filter by subject (OS, DSA, DBMS, CN)
- 💾 Data saved using LocalStorage
- 📱 Responsive and clean UI
- ⚡ Fast and lightweight (no backend required)

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage API
