## 📖 Project Description

Study Resources Hub is a frontend web application designed to help students organize their study materials in a simple and efficient way. It allows users to add, view, search, and filter academic resources such as notes, video lectures, and external learning links.

The main purpose of this project is to reduce the difficulty of managing scattered study resources by providing a centralized and user-friendly interface.

This application is built using React.js and focuses on core frontend development concepts such as component-based architecture, state management, form handling, and LocalStorage for data persistence.

⚠️ Note: The project is currently not deployed due to GitHub Pages configuration limitations with Vite. However, it runs perfectly in the local development environment.
